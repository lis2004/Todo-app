# Todo App - React

To-Do List приложение на React с использованием useEffect, React Router и API.

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение откроется на http://localhost:3000

## Функционал

- **My Todos** - добавление, удаление, фильтрация и поиск задач (сохранение в LocalStorage)
- **API Todos** - загрузка задач с публичного API

## Структура

```
src/
├── components/ - компоненты интерфейса
├── pages/ - страницы приложения
├── App.js - маршрутизация
└── index.js - точка входа
```

## Используемые технологии

- React 18
- React Router v6
- Fetch API
- LocalStorage
