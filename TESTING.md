## 📝 My Todos (Главная страница - /)

### ✅ Добавление задач
1. [ ] Введите текст в поле "Add a new task"
2. [ ] Нажмите кнопку "Add" или Enter
3. [ ] Новая задача появляется в списке

**Проверка:**
- Задача добавляется в список
- Input очищается после добавления
- Задача появляется с id (timestamp)

### ✅ Отметить выполненное
1. [ ] Нажмите на чекбокс рядом с задачей
2. [ ] Текст должен зачеркиваться
3. [ ] Нажмите снова - зачеркивание исчезнет

**Проверка:**
- Класс "completed" добавляется/убирается
- Text-decoration: line-through появляется/исчезает

### ✅ Удаление задач
1. [ ] Нажмите кнопку "Delete" рядом с задачей
2. [ ] Задача удаляется из списка

**Проверка:**
- Задача сразу исчезает из интерфейса
- Filter и search обновляются

### ✅ Фильтрация
1. [ ] Нажмите кнопку "All" - должны видеть все задачи
2. [ ] Нажмите "Active" - видите только невыполненные
3. [ ] Нажмите "Completed" - видите только выполненные

**Проверка:**
- При "Active" видны только задачи с completed: false
- При "Completed" видны только задачи с completed: true
- При "All" видны все задачи

### ✅ Поиск
1. [ ] Введите текст в поле поиска вверху
2. [ ] Список фильтруется в реальном времени
3. [ ] Поиск работает в любом регистре

**Проверка:**
- "молоко" находит "Купить молоко", "Молоко для кота"
- "МОЛОКО" работает так же (регистронезависимый)
- Фильтрация срабатывает при каждом символе

### ✅ LocalStorage (Персистентность)
1. [ ] Добавьте несколько задач
2. [ ] Нажмите F5 (перезагрузка страницы)
3. [ ] Все задачи остаются!

**Проверка:**
- После перезагрузки todos восстанавливаются
- Статус выполнения (completed) сохраняется
- Откройте DevTools → Application → LocalStorage → myTodos

## 🌐 Shared Todos (Страница с API - /shared)

### ✅ Загрузка с API
1. [ ] На странице видно "Loading todos from API..."
2. [ ] Через 1-2 секунды появляется список задач
3. [ ] Всего должно быть 10 задач

**Проверка:**
- Первоначально видно "Loading..."
- Данные загружаются с JSONPlaceholder API
- Количество: ровно 10 tasks (_limit=10)

### ✅ Отображение данных с API
1. [ ] Каждая задача имеет title (текст) и completed (статус)
2. [ ] Примеры: "delectus aut autem", "quis ut et voluptatem"
3. [ ] Некоторые задачи уже выполнены (зачеркнуты)

**Проверка:**
- Данные корректные (от JSONPlaceholder)
- Title отображается правильно
- Статус completed показывает правильно

### ✅ Отметить выполненное (API)
1. [ ] Нажмите чекбокс на задаче с API
2. [ ] Задача зачеркивается/раскрывается

**Проверка:**
- Состояние изменяется в реальном времени
- Это НЕ сохраняется на сервер (read-only)

### ✅ Удаление с API
1. [ ] Нажмите "Delete" на задаче
2. [ ] Задача удаляется из списка

**Проверка:**
- Задача исчезает
- Это локальное удаление (не на сервере)

## 🔍 Проверка кода (DevTools)

### ✅ Console (Консоль)
1. [ ] Откройте DevTools (F12) → Console
2. [ ] Не должно быть ошибок (красные сообщения)
3. [ ] Могут быть warnings (жёлтые)

### ✅ Elements/Inspector
1. [ ] Откройте Elements → найдите root
2. [ ] Структура должна быть чистой и логичной
3. [ ] Проверьте наличие ключей в списках

**Проверка React DevTools:**
1. [ ] Установите React DevTools расширение
2. [ ] Откройте DevTools → Components
3. [ ] Видно дерево компонентов:
   ```
   App
   ├─ Router
   ├─ Navigation
   └─ HomePage / SharedTodosPage
      ├─ AddTodo
      ├─ Search
      ├─ Filter
      └─ TodoList
         └─ TodoItem x10
   ```

### ✅ LocalStorage
1. [ ] F12 → Application → LocalStorage
2. [ ] Выберите localhost:3000
3. [ ] Найдите ключ "myTodos"
4. [ ] Значение должно быть JSON массивом

```json
[
  {"id": 1234567890, "text": "Задача", "completed": false},
  {"id": 1234567891, "text": "Выполненная", "completed": true}
]
```

### ✅ Network
1. [ ] F12 → Network
2. [ ] Перейдите на Shared Todos
3. [ ] Должен быть запрос на `jsonplaceholder.typicode.com`
4. [ ] Status должен быть 200 (успешно)

**URL запроса:**
```
https://jsonplaceholder.typicode.com/todos?_limit=10
```

## 🐛 Частые проблемы

### Проблема: "npm: команда не найдена"
**Решение:** Добавьте Node.js в PATH (смотрите README.md)

### Проблема: Задачи не сохраняются
**Решение:** Проверьте LocalStorage в DevTools → Application

### Проблема: API задачи не загружаются
**Решение:** 
1. Проверьте интернет соединение
2. Проверьте Network вкладку в DevTools
3. Может быть rate limit на API

### Проблема: Навигация не работает
**Решение:** Убедитесь, что React Router правильно импортирован в App.js
